P3 / P4 / E6 / E7 状态入口（在 P2 内部触发）
  1. P3 Skill 编辑器抽屉：进 P2 → 右栏 Skill tab → 点 + 新建 Skill 或 编辑。
  2. P3 Skill 表单校验失败：抽屉里 name 留空 / 含 _ / 含大写 → 点保存看 Alert。
  3. E7 导入解析失败：P2 → Skill tab → 点 导入 SKILL.md（多次点击会随机出现：解析失败 Alert / 重名 ConfirmModal / 成功）。
  4. E6 首次启用须知：P2 → Tool tab → 勾选 联网搜索 → 弹用户须知 Modal；同一用户后续不再弹。
  5. P4 删除确认：P1 卡片 删除 或 P2 顶栏 删除 → 弹二次确认 Modal。
  6. 已删除恢复：P1 → 顶部 已删除 tab → 点卡片 恢复；或点 永久删除 二次确认。
  7. E1 重试机制：P2 试运行区右上 模拟 1 次抖动 / 模拟 3 次失败。
  8. E5 网络断开顶栏：P1 顶栏 模拟离线 按钮（再点一次关闭）。
  9. MCP 占位灰显：P2 → Tool tab 旁的 MCP tab，点击后显示"即将上线"占位。
  10. 移动端响应式：把窗口宽度调到 < 768 px，看导航折叠 + 卡片单列 + FAB。
