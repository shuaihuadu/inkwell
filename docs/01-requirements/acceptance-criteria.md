> 本文档把 [requirements.md](./requirements.md) 第 5 节每条 `AC-XXX-N` 与 [ui-spec.md](./ui-spec.md) 的页面 / 状态 / [user-flow.md](./user-flow.md) 的流程节点显式绑定，并写出可由黑盒手测复现的 **测试动作 → 期望结果**。每行的 `AC` 列可直接被 H4 阶段拓展为 `TC-NNN`。

## 1. 术语

- **测试动作**：用浏览器 + 鼠标 / 键盘可执行的具体操作序列。
- **期望结果**：可观察的 UI 变化或后端状态变化（无需读后端日志）。
- **页面 / 状态**：[ui-spec.md](./ui-spec.md) 第 2 / 3 节的页面编号与状态。
- **流程**：[user-flow.md](./user-flow.md) 的 F / FE 编号。
